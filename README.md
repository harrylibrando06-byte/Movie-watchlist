# Minimalist Todo List

A clean, glassmorphism-style todo list app built with vanilla JavaScript, HTML, and CSS. Add tasks, remove them, search, and pick a random task.

## Features

- **Add Tasks** — Add to the start or end of your list
- **Remove Tasks** — Remove first or last task
- **Search** — Check if a task is already in your list
- **Random Pick** — Can't decide what to do? Let the app choose
- **Clear All** — Reset your entire list
- **Checkboxes** — Mark tasks as done (visual only for now)
- **Glassmorphism UI** — Clean, modern design with blur effects and a cloudy blue theme

## Tech Stack

- HTML
- CSS (Glassmorphism, CSS Variables, Flexbox, Custom Scrollbar)
- Vanilla JavaScript (Arrays, DOM Manipulation, Functions, For Loops)

## What I Learned

- Array methods: `push`, `pop`, `unshift`, `shift`, `includes`, `join`
- Building dynamic HTML with `for` loops and `innerHTML`
- Handling empty states and edge cases
- Glassmorphism CSS effects (`backdrop-filter`, semi-transparent backgrounds)
- CSS custom properties (variables) for theming
- Custom scrollbar styling
- DOM manipulation and event handling
